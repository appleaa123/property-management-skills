# Route Delivery Template

When delivering the final compiled route to the manager via `[MANAGER_CHANNEL]`, always use this exact format.

```text
🗺️ **Client Care Route for [Today's Date]**

✅ **CONFIRMED VISITS (Sorted Geographically)**
*These tenants have replied and confirmed.*

1. [Geographic Identifier (e.g., Zip/Postal Prefix)] - [Full Address]
   Tenant: [Name] | Reason: [Status Flag]
2. [Geographic Identifier] - [Full Address]
   Tenant: [Name] | Reason: [Status Flag]

---

⚠️ **UNCONFIRMED / NO REPLY**
*Manager action required: Please call before visiting.*

* [Geographic Identifier] - [Full Address]
  Tenant: [Name] | Reason: [Status Flag]
* [Geographic Identifier] - [Full Address]
  Tenant: [Name] | Reason: [Status Flag]

```
