# Work Order Template

When drafting a work order for vendor dispatch, always use this strict, clear format.

```text
*** WORK ORDER: [Emergency or Routine] ***

**Property Information**
Address: [Full Property Address]
Tenant Name: [Tenant Name]
Tenant Contact: [Tenant Contact Method/Number]

**Issue Description**
Reported Problem: [Brief summary of the issue]
Original Tenant Report: "[Paste the tenant's exact words in quotes]"
Classification: [Trade/Specialty]

**Dispatch Instructions**
Assigned Vendor: [Vendor Company/Name]
Action Required: Please contact the tenant directly to schedule access and diagnose the issue. Provide an estimate to management before beginning any work exceeding $[Pre-Approval Limit, default $250].

Please reply to confirm receipt of this work order.
```
